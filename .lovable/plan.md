

# 🇺🇸 미국 주식 시장 대시보드

실시간 미국 주식 데이터를 한눈에 볼 수 있는 분석 대시보드를 만듭니다.

## 페이지 구성

### 1. 메인 대시보드 (/)
- **시장 지수 카드**: S&P 500, NASDAQ, DOW 지수 현재가와 변동률을 카드 형태로 상단에 표시 (상승: 초록, 하락: 빨강)
- **주가 차트 섹션**: 종목을 검색/선택하면 일봉·주봉·월봉 캔들/라인 차트 표시 (recharts 활용)
- **관심 종목 리스트**: 종목을 추가/삭제할 수 있는 워치리스트 (localStorage에 저장), 현재가·변동률 실시간 표시
- **뉴스 피드**: 시장 관련 최신 뉴스 목록 (제목, 출처, 시간)

### 2. 종목 상세 페이지 (/stock/:symbol)
- 종목 기본 정보 (이름, 티커, 섹터)
- 가격 차트 (기간 선택 가능)
- 관련 뉴스

## 데이터 소스 & 백엔드
- **Lovable Cloud**를 활성화하여 Edge Function으로 외부 API 호출
- **Alpha Vantage** 무료 API를 사용하여 주가, 지수, 뉴스 데이터 수집
- Edge Function이 API 키를 안전하게 관리하고 데이터를 프론트엔드에 전달
