

# AI 챗봇 기능 구상안

현재 미국 주식 대시보드에 AI 챗봇을 추가하는 3가지 방향을 제안합니다.

---

## Plan A: 주식 어시스턴트 챗봇

대시보드 우측 하단에 플로팅 챗 버튼을 두고, 주식 관련 질문에 답변하는 AI 어시스턴트.

**주요 기능:**
- "AAPL 지금 사도 될까?", "테슬라 최근 뉴스 요약해줘" 같은 자연어 질문
- 관심종목 데이터와 뉴스를 AI 컨텍스트로 전달하여 맥락 있는 답변 생성
- 스트리밍 응답으로 실시간 타이핑 효과

**구현 방식:**
- Lovable AI (Gemini Flash) 기반 Edge Function 1개 추가
- 기존 Finnhub 데이터를 시스템 프롬프트에 포함시켜 실시간 시세 기반 답변
- 대화 기록은 로컬 상태로 관리 (세션 단위)

**장점:** 구현 난이도 낮음, 기존 데이터 활용 극대화
**단점:** 범용 대화에는 약함, 투자 조언 면책 필요

---

## Plan B: 뉴스 요약 + 감성 분석 봇

챗봇보다는 자동화된 AI 분석 패널에 가까운 형태. 뉴스를 AI가 요약하고 감성(긍정/부정/중립)을 판단.

**주요 기능:**
- 관심종목별 뉴스 자동 요약 (3줄 요약)
- 감성 분석 태그 (Bullish / Bearish / Neutral) 자동 부착
- "이 종목의 최근 분위기는?" 버튼 클릭 시 AI 브리핑 생성

**구현 방식:**
- Edge Function에서 뉴스 데이터를 Lovable AI에 전달 → 요약 + 감성 분석 결과 반환
- 뉴스 카드 UI에 요약과 감성 태그 표시
- 대화형이 아닌 원클릭 분석 형태

**장점:** UX가 심플, 실용성 높음, 크레딧 소모 예측 가능
**단점:** 자유 대화 불가, 기능 범위 제한적

---

## Plan C: 풀 챗봇 + 도구 호출 (가장 고급)

대화형 AI가 실시간 시세 조회, 관심종목 추가/삭제, 뉴스 검색까지 직접 수행하는 에이전트형 챗봇.

**주요 기능:**
- "NVDA 관심종목에 추가해줘" → 실제로 추가
- "내 관심종목 전체 수익률 보여줘" → 데이터 조회 후 표로 응답
- "이번 주 가장 많이 오른 종목은?" → 시세 비교 후 답변
- 마크다운 렌더링으로 표/리스트 지원

**구현 방식:**
- Lovable AI의 Tool Calling 기능 활용
- Edge Function에서 도구 정의 (시세조회, 종목검색, 관심종목 관리 등)
- AI가 필요한 도구를 선택 → 실행 → 결과를 자연어로 응답
- DB에 대화 기록 저장 (로그인 사용자 한정)

**장점:** 가장 강력하고 인터랙티브한 경험
**단점:** 구현 복잡도 높음, AI 크레딧 소모 많음, 개발 시간 길음
